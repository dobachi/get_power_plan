# README

このスクリプトは、現在の電源プランを確認するためのものです。

## 実行方法

```powershell
powershell -ExecutionPolicy RemoteSigned -File <スクリプトのディレクトリ>\get_power_plan.ps1
```

`<スクリプトのディレクトリ>` の箇所を置き換えてください。

## 画面ロック解除時に起動する方法

Windowsのタスクスケジューラで設定します。

### 手順

「操作」→「タスクの作成」から新しいタスクを作成します。

* 「全般」タブ
  * 「名前」に任意の名前を入力します
* 「トリガー」タブ
  * 「新規」押下
  * 「タスクの開始」を「ワークステーション アンロック時」にします
* 「操作」タブ
  * 「新規」押下
  * 「プログラム/スクリプト」に「powershell」を入力します
  * 「引数の追加」に「-ExecutionPolicy RemoteSigned -File <スクリプトのディレクトリ>\get_power_mode.ps1」を入力します。
* 「条件」タブ
  * 「コンピューターをAC電源で使用している…」のチェックボックスを外します
